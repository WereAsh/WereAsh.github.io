机器学习

基本术语

​ 在生活中，我们可以根据西瓜的色泽、根蒂、敲声这几个特征来判断一个西瓜的好坏

​ 假如我们要使用机器学期的方法判断西瓜的好坏，并收集到了如表中所示的数据

编号 色泽 根蒂 敲声
1 青绿 蜷缩 混响
2
3
… … … …
  • 我们把所有数据的集合成为一个数据集
  • 把其中每条记录是对于某个对象的描述，成为一个样本
  • 反映对象某方面的性质的事项，如表中的色泽、根蒂等称为属性或特征
  • 从数据中学习模型的过程称为学习或训练，训练过程中使用的样本称为训练样本
  • 有时候，我们为了训练模型还需要样本的结果信息，例如1号西瓜的结果是”好瓜“，我们称这种结果信息为标签。而将一条拥有标签的样本成为一个样例

分类：就是把主句划分为不同的类

回归：就是一些数据，要对这些数据进行线性拟合和非线性拟合

聚类：有一批数据，要把数据聚为几类

工作流程

  1. 选择数据：将数据分割为三组，即训练集和测试集
  2. 构建模型：利用从训练数据中所获得特征构建模型
  3. 验证模型：利用验证集对模型进行验证
  4. 测试模型：利用测试集对模型性能进行测试
  5. 用户使用：对所训练的模型对新的数据进行处理
  6. 调整模型：通过获得更多特征或者对模型的参数进行调整，以提升算法的性能

监督学习

在监督学习中，训练样本的标签（即结果）是已知的，学习目标是通过对样本以及对应标签的学习，得到样本特征与标签之间的内在联系，从而实现对未知样本标签进行预测。

根据样本连续还是离散，连续还是离散，监督学习又可以分为回归和分类

例如我们可以通过西瓜的根蒂、色泽、敲声来训练模型，把西瓜分为好瓜和坏瓜，这是一种分类

而我们通过对过去三年的各地段各房型房价变化推测今年的房价，就是一种回归

监督学习流程

  1. 划分训练集/测试集
  2. 用训练集训练模型
  3. 用模型测试集进行分类
  4. 输出分类结果

举例：

  1. 我们从每个类别随机选取10个样本集，剩余样本作为测试集
  2. 使用训练样本集训练一个分类模型
  3. 使用训练好的模型对测试集样本进行分类
  4. 将分类结果输出

支持向量机

支持向量机（Support Vector Machine SVM）是经典的监督学习算法，它能够根据训练样本的信息，找到一个最优分离超平面

支持向量：支持或者支撑平面上把两类类别划分开来的超平面的向量点

机：机器，便是一个算法

目标：找到一个最优超平面，使得它能够尽可能多的将两类数据点正确地分开，同时使分开的两类数据点距离分类面最远

思路：寻找正例和反例之间具有最大间隔的超平面

我们最终想要达到的目的是，寻找一个如下的超平面（在n维特征空间中）

  • 这就让我们希望超平面应该距离两类数据簇中距超平面最近的两个数据点距离最大
  • 如何找到最合适的分类超平面？依据的原则就是间隔最大化
  • 不偏向任何一方，才能保证双方利益最大化。对于超平面来说，也就是位于两类数据的正中间
  • 不怕偏向任何一类，才能保证离两边数据尽可能远，从而实现间隔最大化。

若输入数据所在的特征空间存在决策边界（decision boundary）的超平面将学习目标按正类和负类分开，并是任意样本的点到平面距离大于等于1（规范化约束）：

对于给定的训练样本，首先要找到距离分类超平面最近的点（支持向量），在通过最大化这些点的间隔来求解

SVM的最终目标是：以间隔最大化为原则找到最合适的那个分类器

SVM核函数

SVM最初提出是针对二分类问题，但是可以方便的扩展到多类问题的划分中去。常用的方法有两种。假设是k分类问题：

  • 逐一鉴别（one-against-all）：系统构造k个SVM，每个SVM分别将某一类别的数据从其他类数据中鉴别出来。也就是每次将其中一个类别的训练数据作为一个类别，其他不属于该类别的训练数据作为另外一个类别
  • 一一区别（one-against-one）：建立k(k-1)/2个SVM，在每两类之间训练一个SVM，把一类与另外一类分开
