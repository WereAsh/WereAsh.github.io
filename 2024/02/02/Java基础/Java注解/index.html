Java注解

javax.validation.constraints

  1. @NotNull

    不能为null，但可以为empty，一般用于判断Integer等基本数据类型，而且被其标注的字段可以使用@size、@Max、@Min对数值进行大小的控制

  2. @NotEmpty

    不能为null，长度必须大于0，一般用于集合类或者数组

  3. @NotBlank

    只能作用在接收的String类型上，不能为null，而且调用trim()后，长度必须大于0。即必须有实际字符
